Optional ionApiUrl

ionApiUrl: string
  • Defined in projects/infor-up/m3-odin/m3/types.ts:41

Gets the Ion Api Url if configured on the server. ION API can only be used in cloud. If the application is a context app (run otside of H5) then a H5 session is needed to get the UserContext with this information. A user context from MNS150MI does not have this information.

Optional isMultiTenant

isMultiTenant: boolean
  • Defined in projects/infor-up/m3-odin/m3/types.ts:48

Gets if H5 is running in a multitenant configuration. It is a multi-tenant environment if the tenant is different from 'infor'.
