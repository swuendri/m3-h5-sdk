Interface IBookmark

Represents an M3 bookmark.

import { IBookmark } from '@infor-up/m3-odin';

Example

This example shows how to define a stateless bookmark definition for the E-panel in CRS610.

private bookmark = {
   program: 'CRS610',
   table: 'OCUSMA',
   keyNames: 'OKCONO,OKCUNO',
   option: '5',
   panel: 'E',
   isStateless: true
} as IBookmark;

Example

This example shows how to clone an existing bookmark definition and set some key values. Note that CONO and DIVI fields can normally be left out since they are resolved automatically using the M3 user context (if available).

var bookmark = Object.assign(this.bookmark);
bookmark.values = {
   OKCONO: '100',
   OKCUNO: 'TEST'
};

The string should include all key names and values in the following pattern?: Key-1,Value-1,Key-2,Value-2,Key-N,Value-N. The values should be URL encoded using the UTF-8 encoding to ensure that that the keys string can be parsed correctly on the BE server.
