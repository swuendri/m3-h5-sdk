Interface IFormService

Represents the M3 form service.

import { IBookmark } from '@infor-up/m3-odin';

The form service can be used to launch stateless requests for bookmarks, search and translation.

getEnvironmentContext

  • getEnvironmentContext(): Observable<IEnvironmentContext>
    • Defined in projects/infor-up/m3-odin/form/types.ts:302

    Gets the environment context information. If there is an existing user context the values will be retreived from the user context. Don't call this method if the application will run within the H5 tab system. The values in the Environment context are also available on the user context if the application is running in a H5 tab.

    Returns Observable<IEnvironmentContext>
